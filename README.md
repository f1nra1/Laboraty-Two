## Task 1: Вывод буквенного паттерна

### Условие:
Задано целое число `N`. Необходимо вывести паттерн из заглавных английских букв в обратном алфавитном порядке, начиная с `N`-й по счёту буквы (от 'A'), по следующему правилу:
- В первой строке выводится одна буква,
- В каждой последующей — на одну букву больше, начиная с той же буквы, что и в первой строке, но в обратном порядке.

### Пример 1:
- **Входные данные**: N = 2  
- **Результат**:
  ```
  B
  B A
  ```

### Пример 2:
- **Входные данные**: N = 4  
- **Результат**:
  ```
  D
  D C
  D C B
  D C B A
  ```

## Task 2: Полив цветов с ограничением

### Условие:
У вас есть грядка с цветами, расположенными в один ряд и пронумерованными от 0 до N-1 слева направо. Каждый цветок требует определённого количества воды. Рядом с грядкой на позиции `x = -1` находится река, где можно наполнить лейку.

Алгоритм полива:
- Начинаем с позиции `-1` (у реки) с полной лейкой.
- Идём слева направо и поливаем цветы.
- Если в лейке не хватает воды для следующего цветка, возвращаемся к реке, набираем воду и продолжаем с того места, где остановились.
- Передвижение осуществляется по горизонтальной оси на 1 шаг за раз (вперёд или назад).
- Необходимо минимизировать количество шагов.

На вход подаётся:
1. Количество цветов `N`
2. Потребности каждого цветка
3. Вместимость лейки

Необходимо вывести минимальное количество шагов, необходимых для полива всех цветов.

### Пример 1:
- **Входные данные**:  
  ```
  5
  2 2 3 3 3
  5
  ```
- **Результат**: 14

### Пример 2:
- **Входные данные**:  
  ```
  6
  1 1 1 4 2 3
  4
  ```
- **Результат**: 30

### Пример 3:
- **Входные данные**:  
  ```
  7
  7 7 7 7 7 7 7
  8
  ```
- **Результат**: 49

## Task 3: Сумма цифр, делящихся на 3

### Условие:
Дан список целых чисел. Необходимо определить сумму **всех цифр**, которые делятся на 3, **во всех числах списка**.

### Пример:
- **Входные данные**: `3 92 102 33`  
- **Результат**: 18

*Пояснение:*  
Цифры из всех чисел: `3, 9, 2, 1, 0, 2, 3, 3`  
Цифры, делящиеся на 3: `3, 9, 0, 3, 3`  
Сумма: `3 + 9 + 0 + 3 + 3 = 18`
